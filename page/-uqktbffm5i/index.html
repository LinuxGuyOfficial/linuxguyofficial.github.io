Checking Palindrome
using Python


What is Palindrome ?

Palindrome is a word or Sequence of Alphabets & Numbers Which if you read forward and Backward they spell same.

For Example: WOW, MALAYALAM, OYO, & Etc, if you read or write them forward or backward they will spell same.

So, there is no Function in python to check if a Given String/Number is A Palindrome or Not, So here are 3 Methods Which You Can use to check if a String is A Palindrome Or Not.

1) Using A 'For Loop'

def isPalindrome(string):
  new_str = ""
  for i in string:
    new_str = i + new_str

  if new_str == string:
    return True
  else:
    return False

string = str(input("Enter Something: "))

if isPalindrome(string.lower()):
  print(f"'{string}' is A Palindrome")
else:
  print(f"'{string}' is Not A Palindrome")


~$ python3 method-1.py
Enter Something: Wow
'Wow' is A Palindrome

~$ python3 method-1.py
Enter Something: HELLO
'HELLO' is Not A Palindrome

So in the first Method what is happening is There is a Function named isPalindrome() which takes a argument (string) and using a 'for loop' it reverses the string and Compares if the Given argument/String is Equal to the new Reversed String.

If the string is Equal to the Reversed String then the function Returns 'True' else it Returns 'False'

2) Slicing That Starts From back

def isPalindrome(string):
  if string == string[::-1]:
    return True
  else:
    return False

string = str(input("Enter Something: "))

if isPalindrome(string.lower()):
  print(f"'{string}' is A Palindrome")
else:
  print(f"'{string}' is Not A Palindrome")


~$ python3 method-2.py
Enter Something: Madam
'Madam' is A Palindrome

~$ python3 method-2.py
Enter Something: Apple
'Apple' is Not A Palindrome

So in this second Method what is happening is the function named isPalindrome() which takes a argument (string) and using 'Slicing' it reverses the string and Compares if the Given argument/String is Equal to the new Revresed String.

If the string is Equal to the Reversed String then the function Returns 'True' else it Returns 'False'


2) Slicing Without Using Any Function

string = str(input("Enter Something: "))

if string.lower() == string.lower()[::-1]:
  print(f"'{string}' is A Palindrome")
else:
  print(f"'{string}' is Not A Palindrome")


~$ python3 method-2.py
Enter Something: radar
'radar' is A Palindrome

~$ python3 method-2.py
Enter Something: Mango
'Mango' is Not A Palindrome

So in this third Method what is happening is we haven't used any type of function which reverses the string but in place of that we are using an 'If' statement where using '[::-1]' we are slicing the string and Comparing if the Given String is Equal to the new Reversed String

If the string is Equal to the Reversed String then this Part of the Code print(f"'{string}' is A Palindrome") At line 4 is Executed else the code at Line 6 print(f"'{string}' is Not A Palindrome") is Executed.


So overall, all of the methods i showed up were doing only one thing Reversing the string & then comparing It
